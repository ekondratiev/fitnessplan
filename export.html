This app uses iTunes to transfer file to your computer on the device that supports file sharing. Export file is being created every time you launch this screen and it is Excel XML document.

File is transferred instantly. You don't need to sync your device to transfer file, but your device needs to be connected to your computer.

To transfer files from your iPod touch, iPhone, or iPad to your computer:

  1. Connect your device to your computer.
  2. In iTunes, select your iPod touch, iPhone, or iPad (below Devices), and then click the Apps tab.
  3. Below File Sharing, from the list on the left, select the FitnessPlan app on your device that you want to transfer files from.
  4. From the list on the right, select the file named fitnessplan3.xls which should be the only one file listed and then click "Save to."
  5. In the window that appears, select the destination where you want the file to be saved, and then click Choose.
